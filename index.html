14.块引用

给引用的文本开始位置都加一个 '>'，  
便可组成一个块引用。在块引用中，可以结合  
其他markdown元素一块使用，比如列表。  
**强调**  
也可以只在第一行加大于号，其他位置不加。  
- 块引用里使用列表，需要和上面的内容隔开一个空行
- 记得加空格哦。

Linux端口占用、进程使用命令

发表于 2018-10-29 | 分类于 Linux

本文主要介绍Linux如何查看端口占用的情况

  • lsof -i:port // 查看某一端口的使用情况。
  • netstat -anp | grep key // 查看所有进程和端口的使用情况。
  • netstat -tunlp | grep key // 用于显示tcp、udp的端口和进程等相关情况。
  • ps -aux | grep key // 用于查看进程的使用情况
